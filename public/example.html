Graf.JS
Create and perform operations on customizable tables using this small Javascript library.
Documentation

Examples

Using this library there are three preset table styles avalible to be used.

We can also change table dimensions, column label fonts, title fonts and title font size.

We can also get tables to display the table based on the rows we wish to look at. In this case we will look at row index 0 to 2 with the same data based on the previous table.

Notice how the table stretches the rows inorder to maintain the specified table height.

This library also allows the user to add and remove rows. Usually the user can choose which row to remove and what rows to add. For this sample we will only remove the first row and add premade data to show that the library is capable of the task.

This library also allows the user to add and remove columns as well. For this sample we will add an remove a premade column.
