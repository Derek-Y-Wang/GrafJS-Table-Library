Graf.JS Documentation
Main

Getting Started:

    1. Download Graf.JS from https://github.com/csc309-fall-2020/js-library-wangder1
    2. Drag graf.js into the same directory as your HTML or Javascript.
    3. Add <script type="text/javascript" src="graf.js"></script> in your HTML file.
    4. Create a <div> in your HTML file with an specific ID such as <div id="sample">
    5. You must now initialize the Graf object in your Javascript file. Initialize the Graf object with the ID of where you would want to insert a table (id="sample" for example.)

Content:

    Initialization:

    new Graf({
    data: [[value,...],...](required),
    columns: [value,...](required),
    anchor: String(required),
    title: String(required),
    style: String,
    height: String,
    width: String,
    titleSize: String,
    columnFont: String,
    titleFont: String
    });
  • data: User must input the data in the form of a list of lists, in the form of [[value,...],...]. Each inner list corresponds to a column in the table.
  • columns: User must input the data in the form of a list, in the form of [value,...]. Each value in the list is the label for a column.
  • anchor: User must input the ID element in the form of a String, this will allow the library to know where to place the table in the HTML file.
  • title: User must input the title of the table in the form of a String, this will allow the library to generate a title for the table and create a table ID.
  • style: User must input the title of the table in the form of a String, user has a choice between three styles,
  • linear: This is the default style.
  • clear: No borders between data cells.
  • boxy: Boxed data cells.
  • height: User is able to set the height of the table in the form a String. Must be in CSS format such as "40px".
  • width: User is able to set the width of the table in the form a String. Must be in CSS format such as "40px".
  • titleSize: User is able to set the size of the of the title in the form a String. Must be in CSS format such as "40px".
  • columnFont: User is able to set the size of the of the font of the column labels in the form a String. Must be in CSS format such as "Times New Roman".
  • titleFont: User is able to set the size of the of the font of the title in the form a String. Must be in CSS format such as "Times New Roman".

    Functions

  • makeFullTable(): Generates the table based on data and other given values upon initialization.
  • makeTableByRows(start=Integer, end=Integer): Generates the table rows within the given indexes on data and other given values upon initialization. Default start and end is null and leaving them empty will default to the last or first row respectively.
  • addColumn(col_data:List, col_name:String): Adds a new column to an existing table. col_data is a list containing all the elements in the new column and col_name will be the new column label.
  • addRow(row_data:List): Adds a new row to an existing table. row_data is a list containing all the elements in the new row.
  • removeColumn(col_name:Integer/String): Removes a column from the existing table. Parameter can be either a String or Integer. Removes column by label name if String,removes column by index if Integer.
  • removeRow(row_index:Integer): Removes a row from the table based on inputed row index.
  • fillNullColMean(): Should be called before creating the table. Will replace all empty data cells with the mean of the column.
  • fillNullColZeros(): Should be called before creating the table. Will replace all empty data cells with zero.
  • fillNullCol(value): Should be called before creating the table. Will replace all empty data cells with any value.
  • getColumnNames(): Returns all column labels in the form of a List.
  • printSelectRow(index:Integer): Returns all values at the indexed row in the form of a List.
  • printSelectColumn(index:Integer): Returns all values at the indexed column in the form of a List.
